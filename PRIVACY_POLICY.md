# 隐私政策

**最后更新: 2025年12月12日**

[English](./PRIVACY_POLICY_EN.md) | 中文

## 简介

OneBookmark（"本扩展"）是一款将书签同步到 GitHub Gist 的浏览器扩展。本隐私政策说明我们如何处理您的数据。

## 我们访问的数据

| 数据类型 | 用途 |
|----------|------|
| **浏览器书签** | 读取并同步到 GitHub Gist |
| **GitHub Personal Access Token** | 用于 GitHub API 身份验证 |
| **Gist ID** | 标识您的备份位置 |

## 数据使用方式

- **书签同步**: 仅当您主动执行同步操作时，书签才会上传到您自己的 GitHub Gist。
- **本地存储**: 设置信息（Token、Gist ID）存储在浏览器的扩展存储中。
- **无服务器**: 我们不运营任何后端服务器。数据仅在您的浏览器和 GitHub 之间直接传输。

## 数据共享

- **我们不收集、存储或向任何第三方传输您的数据。**
- 您的书签仅存储在您个人的 GitHub Gist 账户中。
- 我们无法访问您的 GitHub Token 或书签数据。

## 数据安全

- 您的 GitHub Token 存储在浏览器的安全扩展存储中。
- 与 GitHub API 的所有通信均使用 HTTPS 加密。
- 建议使用最小权限的 Token（仅需 `gist` 权限）。

## 您的权利

您可以：
- **查看** 扩展设置中存储的所有数据
- **删除** 通过卸载扩展删除所有扩展数据
- **撤销** 随时在 GitHub 设置中撤销 Token 访问权限

## 第三方服务

本扩展与以下服务交互：
- **GitHub API** - 受 [GitHub 隐私政策](https://docs.github.com/zh/site-policy/privacy-policies/github-general-privacy-statement) 约束

## 政策变更

我们可能会更新本政策。变更将发布在我们的 GitHub 仓库中。

## 联系方式
